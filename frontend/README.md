### Prerequisites
- Node.js & npm
- Python 3.8+
- [Google Gemini API Key](https://aistudio.google.com/)

### Setup

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd LocalInsight
   ```

2. **Frontend Setup**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. **Backend Setup**
   ```bash
   cd ../backend
   pip install -r requirements.txt
   # Create a .env file and add:
   # GEMINI_API_KEY=your_key_here
   python app.py
   ```
